# Massas Santo Antonio

Site institucional da **Massas Alimenticias Santo Antonio**, empresa familiar com mais de 50 anos de tradicao na producao artesanal de bolachas e massas em Itabaiana, Sergipe.

## Acesse o site

[diggo1926.github.io/site-massas-santo-antonio](https://diggo1926.github.io/site-massas-santo-antonio/)

## Sobre o Projeto

Site institucional completo, com apresentacao da empresa, catalogo de produtos, processo de producao, area de distribuicao e formulario de contato com integracao ao WhatsApp.

## Tecnologias Utilizadas

- **HTML5** — estrutura semantica e acessivel
- **CSS3** — estilos personalizados com variaveis CSS
- **JavaScript** — interatividade e menu mobile
- **Tailwind CSS** — utilitarios de estilo responsivo
- **Lucide Icons** — icones modernos e leves
- **Google Fonts** — Playfair Display e Source Sans 3
- **GitHub Pages** — hospedagem gratuita

## Estrutura de Arquivos

```
site-massas-santo-antonio/
├── index.html       # Pagina principal
├── css/
│   └── style.css    # Estilos customizados
├── js/
│   └── script.js    # Scripts de interatividade
└── imagens/         # Imagens e icones do site
```

## Funcionalidades

- Navegacao fixa com menu desktop e mobile responsivo
- Secao Hero com animacoes suaves
- Cards de produtos com efeito hover
- Secao de processo de producao em 4 etapas
- Area de distribuicao regional
- Formulario de contato integrado ao WhatsApp
- Footer com links e horario de funcionamento
- Meta tags Open Graph para compartilhamento em redes sociais
