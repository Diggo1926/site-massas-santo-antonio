# 🍪 Massas Santo Antônio

Site institucional da **Massas Alimentícias Santo Antônio**, empresa familiar com mais de 50 anos de tradição na produção artesanal de bolachas e massas em Itabaiana, Sergipe.

## 🔗 Acesse o site

👉 [diggo1926.github.io/site-massas-santo-antonio](https://diggo1926.github.io/site-massas-santo-antonio/)

## 📋 Sobre o Projeto

Site institucional completo, com apresentação da empresa, catálogo de produtos, processo de produção, área de distribuição e formulário de contato com integração ao WhatsApp.

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura semântica e acessível
- **CSS3** — estilos personalizados com variáveis CSS
- **JavaScript** — interatividade e menu mobile
- **Tailwind CSS** — utilitários de estilo responsivo
- **Lucide Icons** — ícones modernos e leves
- **Google Fonts** — Playfair Display e Source Sans 3
- **GitHub Pages** — hospedagem gratuita

## 📂 Estrutura de Arquivos

```
site-massas-santo-antonio/
├── index.html       # Página principal
├── css/
│   └── style.css    # Estilos customizados
├── js/
│   └── script.js    # Scripts de interatividade
└── imagens/         # Imagens e ícones do site
```

## ✨ Funcionalidades

- Navegação fixa com menu desktop e mobile responsivo
- Seção Hero com animações suaves
- Cards de produtos com efeito hover
- Seção de processo de produção em 4 etapas
- Área de distribuição regional
- Formulário de contato integrado ao WhatsApp
- Footer com links e horário de funcionamento
- Meta tags Open Graph para compartilhamento em redes sociais
